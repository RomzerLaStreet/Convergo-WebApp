# Convergo

Trouve la date qui convient à tout le monde.

## Technologies

- **Frontend** : Next.js 16 (App Router)
- **Backend** : Convex
- **Style** : Tailwind CSS 4

## Installation

```bash
npm install
```

## Développement

Lancer Convex en mode développement :
```bash
npx convex dev
```

Lancer l'application Next.js :
```bash
npm run dev
```

## Authentification

L'application utilise un système d'authentification locale (`localStorage`) pour identifier les utilisateurs sans nécessiter de compte externe.
